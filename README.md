# TemAI Lite Auth Shell

Minimal React + Vite shell for TemAI Lite with Supabase Auth.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy env file and fill in Supabase values:
   ```bash
   cp .env.example .env
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```

## What is included

- Login screen for unauthenticated users.
- Simple app shell for authenticated users.
- Top nav with `Dokumenter` and `Last opp`.
- Signed-in user email and logout button.
- No backend API calls.
